<!-- GETTING STARTED -->
## Getting Started
### Prerequisites

Prerequisites :  - Install Angular with ```sh npm install -g @angular/cli ```
                 - Open the project in IDE and do ```sh npm i```
                 - Lunch project in local with ```sh ng serve```
